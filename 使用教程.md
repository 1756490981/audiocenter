# AudioCenter 使用教程

## 一、软件简介

AudioCenter 是一款 Windows 音频控制面板软件，专为音乐制作人、录音师和音频爱好者设计。它集成了音量控制、设备管理、音频配置备份、Studio One ASIO 一键修复等功能，让你在一个软件里完成所有音频相关操作。

---

## 二、安装与启动

1. 双击 `AudioCenter.exe` 启动软件
2. 首次启动会弹出**用户账户控制**窗口，点击 **"是"** 允许（软件需要管理员权限才能控制音频设备）
3. 等待几秒，音频服务启动完成后即可使用

> 注意：软件需要**管理员权限**运行，否则无法控制音频设备。

---

## 三、界面说明

软件采用左侧边栏导航，共 5 个功能页面：

| 图标 | 名称 | 功能 |
|------|------|------|
| 🎵 音量 | 音量混合器 | 控制每个应用的音量和输出设备 |
| 🎧 播放 | 播放设备 | 管理扬声器、耳机等播放设备 |
| 🎤 录制 | 录制设备 | 管理麦克风等录音设备 |
| 🔧 Studio | Studio One 修复 | 一键修复 Studio One ASIO 配置 |
| 💾 配置 | 配置管理 | 备份/恢复所有音频设置 |

## 四、功能详解

### 4.1 音量混合器（🎵 音量）

**主音量控制**
- 顶部的滑块控制电脑的主音量
- 点击 🔊/🔇 图标可以静音/取消静音

**每个应用独立控制**
- 下方列出所有正在播放音频的应用（如网易云音乐、QQ音乐、浏览器等）
- 每个应用都有：
  - 🔊/🔇 静音按钮
  - 音量滑块（0-100）
  - 输出设备选择下拉框（可以把不同应用路由到不同声卡）

**使用场景**
- 把音乐播放器路由到外置声卡，游戏声音留在耳机
- 录音时把伴奏软件静音，只录麦克风
- 直播时分别控制不同应用的音量

### 4.2 播放设备（🎧 播放）

**查看设备列表**
- 显示所有播放设备（声卡、耳机、扬声器等）
- 绿色边框 = 当前默认设备
- 蓝色按钮 = 设为默认

**设为默认设备**
- 点击设备卡片上的 **"设为默认"** 按钮
- 默认设备会用蓝色边框高亮显示

**调节设备音量**
- 每个设备下方有音量滑块
- 点击 🔊/🔇 可以单独静音某个设备

**调整音频格式**
- 右键点击设备 → 选择 **"调整格式"**
- 可以设置：采样率（44100/48000/96000/192000 Hz）、位深度（16/24/32 bit）、通道（单声道/立体声）
- 内置常用预设：CD 音质、DVD 音质、录音室、高解析度、发烧级

**启用/禁用设备**
- 右键点击设备 → 选择 **"禁用"** 或 **"启用"**
- 已禁用的设备会折叠显示在底部

### 4.3 录制设备（🎤 录制）

功能与播放设备基本相同，管理麦克风等录音设备：
- 设为默认录音设备
- 调节录音音量
- 调整录音格式
- 启用/禁用设备
- 侦听功能（把麦克风声音路由到扬声器，实时监听）

### 4.4 Studio One 修复（🔧 Studio）

**这个功能是给 Studio One 用户准备的**，解决 ASIO 驱动配置问题。

**选择版本**
- 顶部下拉框选择要操作的 Studio One 版本
- 默认"自动"选择最高版本

**ASIO 驱动列表**
- 自动扫描系统中所有 ASIO 驱动
- 绿色 [在线] = 驱动已安装且可用
- 灰色 [离线] = 驱动未安装或不可用
- 点击 **"选中"** 按钮选择你要用的声卡

**设置采样率和缓冲区**
- 采样率：44100 / 48000 / 96000 Hz
- 缓冲区：64 / 128 / 256 / 512 / 1024 samples
- 缓冲区越小延迟越低，但对 CPU 要求越高

**一键修复并重启**
1. 选中你要用的 ASIO 声卡
2. 设置采样率和缓冲区
3. 点击 **"一键修复并重启 Studio One"**
4. 软件会自动：
   - 关闭正在运行的 Studio One（先发 WM_CLOSE 让你保存工程）
   - 写入 AudioEngine.settings 配置文件
   - 重新启动 Studio One

**仅修复（不重启）**
- 只写入配置文件，不关闭/重启 Studio One
- 适合 Studio One 没有运行的时候使用

**重置音频配置**
- 删除 AudioEngine.settings 文件
- 下次启动 Studio One 会进入音频设置向导

### 4.5 配置管理（💾 配置）

**备份当前设置**
1. 在输入框中输入配置名称（如"录音配置"、"直播配置"）
2. 点击 **"💾 保存"**
3. 保存的内容包括：
   - 所有设备音量和静音状态
   - 默认播放/录制设备
   - 每个应用的输出设备路由
   - Studio One ASIO 配置（声卡、采样率、缓冲区）

**恢复配置**
- 找到你要恢复的配置，点击 **"恢复"** 按钮
- 所有设置会一键恢复

**导出/导入**
- **导出**：把配置文件保存到指定位置（.json 文件），可以发给别人或备份
- **导入**：从文件导入配置

**一键最大音量**
- 点击 **"🔊 所有音量调到 100"**
- 把所有设备和应用的音量拉满

## 五、系统托盘

- 软件启动后会在**右下角系统托盘**显示图标
- 点击关闭按钮不会退出，而是**最小化到托盘**
- 右键托盘图标可以：
  - **显示窗口** — 重新打开主界面
  - **退出** — 完全关闭软件

## 六、常见使用场景

### 场景 1：直播时分别控制音量
1. 打开音量混合器
2. 把游戏声音路由到耳机，音乐路由到外置声卡
3. 分别调节各应用音量，保存为"直播配置"

### 场景 2：录音前调整设备
1. 去录制设备 tab，把声卡设为默认录音设备
2. 去播放设备 tab，把监听耳机设为默认播放设备
3. 调整格式为 48000Hz / 24bit
4. 保存为"录音配置"

### 场景 3：Studio One ASIO 修复
1. 打开 Studio One tab
2. 选中你的声卡（如 Universal Audio、MIDIPLUS 等）
3. 设置采样率 48000、缓冲区 128
4. 点击"一键修复并重启"

### 场景 4：配置备份
1. 把所有设置调好后，去配置 tab 保存
2. 软件更新或系统重置后，一键恢复

## 七、快捷操作

| 操作 | 说明 |
|------|------|
| 最小化到托盘 | 点击标题栏 ─ 按钮或关闭按钮 ✕ |
| 恢复窗口 | 双击托盘图标或右键→显示窗口 |
| 刷新设备列表 | 各页面自动刷新，也可手动点击 ↻ |
| 右键菜单 | 播放/录制设备支持右键操作 |

---

## 八、注意事项

1. **管理员权限**：软件必须以管理员身份运行
2. **Studio One 修复**：会先关闭 Studio One，请确保已保存工程
3. **设备热插拔**：USB 声卡插拔后会自动刷新设备列表
4. **配置文件位置**：保存在 `%LOCALAPPDATA%\AudioCenter\profiles\`
